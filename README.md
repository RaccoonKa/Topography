# 🗺 Military Topography Training Suite

A set of training programs designed for university students. The purpose of this software is to help students master military topography skills through interactive calculations and training.

## 📋 Package contents

| Program | Appointment |
| :--- | :--- |
| **`datacwtopograf`** | A simulator for working with coordinates and azimuths. It allows you to calculate directional angles, magnetic azimuth, and perform transformations between geographical and plane-rectangular coordinates. |
| **`datalyali`** | A program for determining the true azimuth, magnetic azimuth and directional angles. |
| **`datatopograf`** | The main training module on military topography to consolidate theoretical knowledge on determining geographical and plane-rectangular coordinates in practice. |

## 🚀 Software Features

*   **Interactive learning:** The programs allow students to independently perform calculations and test their knowledge.
*   **Topographic accuracy:** The algorithms ensure the correct execution of engineering and geodetic tasks necessary for training.
*   **Practical focus:** The toolkit is intended for use in the educational process in the disciplines of military training.

## 🛠 Technical information

*   **Development language:** Pascal
*   **Wednesday:** Delphi

## 📥 Launch

1.  To work with the source code, open the corresponding `.dproj` file in the Delphi environment.
2.  For quick use, run the executable `.exe` file from the folder of the required module.
3.  The programs do not require complex configuration and are ready for use immediately after launch.
